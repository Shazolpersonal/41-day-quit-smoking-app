# Feature Graphic Design Guide

## Overview
The feature graphic is a banner image displayed at the top of your Play Store listing. It's one of the most important visual elements for attracting users.

## Technical Specifications

### Required Dimensions
- **Size**: 1024 x 500 pixels
- **Format**: PNG or JPEG
- **File size**: Maximum 1 MB
- **Color space**: RGB
- **Aspect ratio**: 2.048:1

### Important Notes
- This is **required** for featured placement
- Displayed prominently on Play Store
- First visual element users see
- Must be high quality and professional

## Design Guidelines

### Safe Area
- **Text safe zone**: Keep important text within 924 x 400 px center area
- **Edge margins**: 50px from all edges
- **Reason**: Different devices may crop edges

### Visual Hierarchy
1. **App name/logo** (most prominent)
2. **Key message** (secondary)
3. **Visual elements** (supporting)

### Color Scheme
Use app's brand colors:
- **Primary**: Green (#4CAF50) - Islamic, health, growth
- **Secondary**: Teal (#009688) - calm, trust
- **Accent**: Orange (#FF9800) - energy, motivation
- **Background**: White or light gradient

## Content Elements

### Must Include
1. **App name**: "৪১ দিনে ধূমপান ত্যাগ"
2. **Tagline**: Short, compelling message
3. **Visual element**: Icon, illustration, or imagery
4. **Branding**: Consistent with app design

### Optional Elements
- App icon
- Key features (icons)
- Call to action
- Decorative elements
- Islamic motifs

## Typography

### Bengali Text
- **Font**: Kalpurush, SolaimanLipi, or Noto Sans Bengali
- **App name size**: 72-96px
- **Tagline size**: 36-48px
- **Weight**: Bold for app name, Regular for tagline
- **Color**: White or dark (high contrast)

### English Text
- **Font**: Roboto, Open Sans, or Inter
- **Size**: 24-36px (smaller than Bengali)
- **Weight**: Regular or Medium
- **Color**: 70% opacity of main text color

### Text Effects
- **Shadow**: Subtle drop shadow for readability
- **Stroke**: Optional outline for contrast
- **Gradient**: Can be used for modern look
- **Avoid**: Excessive effects, hard to read fonts

## Visual Elements

### App Icon
- **Size**: 200-250px
- **Position**: Left side or center
- **Style**: Match app icon design
- **Effect**: Subtle shadow or glow

### Illustrations
- **Style**: Flat, modern, Islamic-inspired
- **Elements**: 
  - Mosque silhouette
  - Islamic patterns
  - Health symbols
  - Progress indicators
  - Calendar/days
  - Smoke-free symbols

### Icons
- **Size**: 80-120px
- **Style**: Line or filled
- **Color**: Monochrome or brand colors
- **Spacing**: Even distribution

### Background
- **Solid color**: Clean and professional
- **Gradient**: Modern and dynamic
- **Pattern**: Subtle Islamic patterns
- **Image**: Blurred or low opacity

## Design Process

### Step 1: Concept
1. Choose design concept
2. Sketch layout
3. Select colors
4. Choose typography

### Step 2: Design
1. Create artboard (1024 x 500 px)
2. Add background
3. Place text elements
4. Add visual elements
5. Apply effects

### Step 3: Refine
1. Check safe area
2. Test readability
3. Adjust spacing
4. Fine-tune colors
5. Add final touches

### Step 4: Export
1. Save as PNG (preferred) or JPEG
2. Optimize file size (under 1 MB)
3. Check quality
4. Create variations

## Variations

Create multiple versions for:
- **A/B testing**: Test which performs better
- **Seasonal**: Ramadan, special occasions
- **Campaigns**: Different marketing messages
- **Languages**: Bengali-focused vs English-focused

## Common Mistakes

### Avoid ❌
- Too much text
- Low resolution images
- Unreadable fonts
- Cluttered design
- Inconsistent branding
- Copyright violations
- Poor color contrast
- Ignoring safe area
- Generic stock photos
- Unprofessional appearance

### Do ✅
- Keep it simple
- Use high quality assets
- Ensure readability
- Maintain brand consistency
- Use original graphics
- High contrast
- Respect safe zones
- Professional design
- Test on devices
- Get feedback

## Best Practices

### Design Principles
1. **Simplicity**: Less is more
2. **Clarity**: Clear message
3. **Consistency**: Match app branding
4. **Quality**: Professional appearance
5. **Relevance**: Reflect app purpose

### Marketing Principles
1. **Value proposition**: Show benefits
2. **Target audience**: Appeal to users
3. **Differentiation**: Stand out
4. **Call to action**: Implicit or explicit
5. **Emotional connection**: Inspire action

## Final Tips

1. **First impressions matter**: This is often the first thing users see
2. **Mobile-first**: Most users browse on mobile
3. **Test thoroughly**: Check on different devices
4. **Keep it updated**: Refresh periodically
5. **Track performance**: Monitor install rates
6. **Iterate**: Continuously improve based on data

  - Shadows
  - Glows
  - Overlays
