# Screenshot Guide for Play Store

## Overview
This guide provides specifications and recommendations for creating screenshots for the Google Play Store listing.

## Screenshot Requirements

### Technical Specifications
- **Format**: PNG or JPEG
- **Minimum dimensions**: 320px
- **Maximum dimensions**: 3840px
- **Aspect ratio**: Between 16:9 and 9:16
- **Recommended size**: 1080 x 1920 px (portrait) or 1920 x 1080 px (landscape)
- **Maximum file size**: 8 MB per screenshot
- **Number of screenshots**: 2-8 (minimum 2 required)

### Recommended Approach
- **Orientation**: Portrait (matches app usage)
- **Size**: 1080 x 1920 px
- **Count**: 6-8 screenshots
- **Language**: Bengali with English subtitles

## Design Guidelines

### Visual Style
- **Clean and modern**: Use the app's actual interface
- **High contrast**: Ensure text is readable
- **Consistent branding**: Use app colors and fonts
- **Real content**: Show actual features, not mockups

### Text Overlays
- **Font**: Use clear, readable Bengali and English fonts
- **Size**: Large enough to read on small screens
- **Color**: High contrast with background
- **Position**: Bottom or top, consistent across all screenshots

### Device Frames (Optional)
- Use Android device frames for context
- Keep frames minimal and clean
- Ensure they don't distract from content

### Localization
- Primary language: Bengali
- Secondary language: English (as subtitle)
- Use culturally appropriate imagery
- Show Bengali interface

## Screenshot Creation Process

### Method 1: Real Device Screenshots
1. Install app on Android device
2. Navigate to each screen
3. Take screenshots using device
4. Transfer to computer
5. Add captions and polish

### Method 2: Emulator Screenshots
1. Run app in Android Studio emulator
2. Set emulator to 1080 x 1920 resolution
3. Navigate to each screen
4. Take screenshots
5. Add captions and polish

### Method 3: Design Tool
1. Use Figma, Sketch, or Photoshop
2. Import app screenshots
3. Add device frames
4. Add captions and overlays
5. Export at correct dimensions

## Caption Design

### Template
```
┌─────────────────────────────────┐
│                                 │
│     [App Screenshot]            │
│                                 │
│                                 │
│                                 │
├─────────────────────────────────┤
│  Bengali Caption (Large)        │
│  English Caption (Small)        │
└─────────────────────────────────┘
```

### Caption Styling
- **Bengali text**: 
  - Font: Kalpurush, SolaimanLipi, or similar
  - Size: 48-60px
  - Weight: Bold
  - Color: White or app primary color

- **English text**:
  - Font: Roboto or similar
  - Size: 32-40px
  - Weight: Regular
  - Color: Light gray or white (70% opacity)

### Background
- Semi-transparent gradient at bottom
- Dark overlay for light screenshots
- Light overlay for dark screenshots
- Ensure text readability
